# OS-Process-Scheduler
Implement a process scheduler controlled by the Operating System.

# Environment

Ubuntu 22.04.4 LTS, 64-bit
GCC compiler

# Execution Command

```bash
gcc -g -Wall -Wextra -o process15 process15.c
./process15

gcc -g -Wall -Wextra -o process30 process30.c
./process30

```
